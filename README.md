# 📖 Overview

**ResponSync** is a full-stack emergency response and incident management platform designed to streamline communication between citizens, emergency responders, and administrative authorities.

The platform enables users to report incidents, monitor emergency requests, manage response teams, and improve coordination through a modern, scalable web application.

---

# ✨ Features

* 🚨 Real-time incident reporting

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/amanantuley/ResponSync.git
```

Navigate to the project directory

```bash
cd ResponSync
```

---

## Backend Setup

```bash
cd backend
npm install
npm start
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

## Environment Variables

Create a **.env** file inside the backend directory.

```env
PORT=5000
MONGODB_URI=your_database_url
JWT_SECRET=your_secret_key
```
